# 離乳食アレルギーチェック

このフォルダには、Androidアプリ版とWebアプリ版の両方があります。

## Webアプリ版

AndroidとiOSのブラウザで使う想定のWeb版を `docs` に追加しました。

- 公開先: GitHub Pages
- 認証/DB: Supabase
- 管理者ログイン対応
- 管理者によるユーザ追加対応
- 管理者による共通の期、分類、食材マスタ編集対応
- 家庭ごとの食材マスタ追加、修正対応
- 子供、記録、メモ、反応あり、PDF/CSV出力対応

セットアップ手順は [WEB_DEPLOY.md](WEB_DEPLOY.md) を見てください。

## Androidアプリ版

Kotlin + Jetpack Compose + Roomで作成したAndroid Studio向けプロジェクトです。

## 実装済み機能

- 初回起動時の子供登録
- 子供マスタの追加、修正、削除
- 子供が複数いる場合のトップ画面ドロップダウン切替
- 初期、中期、後期の横スワイプ切替
- 食材ごとの1回目、2回目の日付入力
- 日付入力時のカレンダー表示、当日デフォルト
- 1回目、2回目それぞれのメモ入力、修正、閲覧
- メモごとの「反応あり」フラグ
- 未実施、1回目済、2回目済、完了、反応ありの色分け
- 食材検索
- 分類フィルタ
- 食材マスタの新規登録、修正
- 食材コード7桁の自動採番
- PDF出力
- CSV出力

## 食材コード

7桁で管理します。

```text
期2桁 + 分類2桁 + 通し番号3桁
```

例: `0101001` は、初期、穀類、通し番号001です。

## 写真から初期登録した食材

写真から読み取れた範囲を初期データとして登録しています。

- 初期 5～6か月
  - 穀類: 米、うどん、そうめん、お麩、パン(卵不使用品可)
  - 野菜: 人参、玉ねぎ、じゃがいも、大根、小松菜、ほうれん草、チンゲンサイ、とまと、かぶ、かぼちゃ、キャベツ、白菜、さつまいも
  - 魚介類: たら、かれい
  - 大豆類: 豆腐
  - その他: 粉ミルク、昆布だし、麦茶
- 中期 7～8か月
  - 穀類: スパゲティ、マカロニ、パン粉
  - 野菜: きゅうり、なす、ブロッコリー、わかめ、さやいんげん
  - 果物: りんご、バナナ
  - 魚介類: 鮭、ツナ(水煮缶)、しらす干し、かつお節
  - 肉類: 鶏肉(胸・ささみ)、鶏ひき肉
  - 大豆類: 大豆(水煮缶)、高野豆腐、きなこ、豆乳
  - 調味料: 砂糖、塩、味噌、しょうゆ、片栗粉、ケチャップ
  - 乳製品: チーズ、バター、牛乳(調理用)
  - その他: サラダ油、焼きのり・青のり、寒天

後期の写真は確認できなかったため、初期データは空です。アプリ内の食材マスタから追加できます。

## 開き方

Android Studioでこのフォルダを開いてGradle Syncを実行してください。`local.properties` には、この端末のAndroid SDKパスを設定済みです。

この作業環境ではAndroid Studio同梱JBRとAndroid SDKは見つかりましたが、GradleコマンドまたはGradle Wrapperがないため、こちらではビルド実行までは確認できていません。
